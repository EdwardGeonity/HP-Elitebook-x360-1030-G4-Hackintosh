# HP-Elitebook-x360-1030-G4-Hackintosh
HP Elitebook x360 1030 G4 Hackintosh


BIOS version: 1.28

Tested on Ventura. Never booted windows from EFI

Working:
- eDP Display with link fix
- Brightness Control (Except Privacy screen mode)
- Battery
- Headphones, Speakers
- Web Camera
- Bluetooth
- Wifi
- USB ports
- IGPU (yet need some fixes for HDMI and TBT ports)
- Sleep
- Wake
- Sleep\Wake with Lid
- Touchscreen
- Touchpad


Not working:
- Internal mic
- Sound keys and power button on case
- eMMC


Some improvements (soldering):
- Increase or decrease soldered RAM
- Enable or disable access to eMMC
